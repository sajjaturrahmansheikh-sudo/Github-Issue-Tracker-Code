## Question 2- What is the spread operator (...)?

Ans : The spread operator (...) in javascript is used to expand elements of an array , object , 
      or iterable into individual elements. It is very usefull for copying , merging, or passing value.



## Question 3- What is the difference between map(), filter(), and forEach()?

Ans : In javascript map(), filter(), forEach() are array methods used to loop through array but they 
      behave differently and used for different purpose.

      forEach() just loop through the array and executes a function for each element is does not return a new array.
      and map() create a new array by transforming each element.
      and filter() creates a new array with elements that pass a condition.
